Экзаменационный проект на Laravel.

Запуск веб приложения:

Настройка .env

DB_CONNECTION=mysql
// Тип соединения

DB_HOST=127.0.0.1
// Хост

DB_PORT=3306
// Порт

DB_DATABASE=my_db
// Имя Базы Данных

DB_USERNAME=root
// Имя пользователя

DB_PASSWORD=root1234
// Пароль к БД

Подготовка к запуску (установка зависимостей): 
composer install

Миграция Базы данных:
php artisan mirgate

Запуск сервера артисан:
php artisan serve

веб приложение откроется по пути:
http://127.0.0.1:8000/
